## âœ¨ Features

- **3D Immersion**: Integrated `Three.js` scenes using `react-three-fiber` for stunning backgrounds and interactive elements like the "Profile Cube".
- **Dynamic Animations**: Powered by `framer-motion` for smooth page transitions, scroll reveals, and micro-interactions.
